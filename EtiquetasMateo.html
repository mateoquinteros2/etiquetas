Etiquetas HTML

A continuación hay una lista de todos los elementos en el estándar HTML hasta la versión 5, junto con una descripción breve acerca de su uso/propósito. Hacer clic en cualquier elemento de esta lista te llevará a una página con una descripción detallada del mismo, algunos ejemplos de uso y una lista de todos los atributos que soporta.
Deberías tener en cuenta que los elementos más nuevos, como aquellos agregados en HTML5, pueden tener un soporte parcial o no ser soportados en absoluto en algunos navegadores. Esto cambiará con el tiempo, pero mientras tanto, si estás diseñando un sitio que necesita verse correctamente en todos los navegadores, tal vez quieras evitar usarlos o reemplazarlos con otras alternativas como los scripts. Todas las características de HTML5 con soporte incompleto de los navegadores están correctamente marcadas en rojo y acompañadas por mensajes de alerta.

<a>

Principalmente utilizado como hipervínculo.

Atributos

Atributos globales
download: Indica que el hipervínculo es utilizado para descargar un recurso.
href: La URL de un recurso asociado.
hreflang: Especifica el lenguaje del recurso asociado.
media: Da un indicio del tipo de medio para el cual el recurso vinculado fue creado.
ping: Indica si se enviara un ping o notificacion a la URL del recurso cuando el usuario utlize el vinculo.
rel: Especifica la relación entre el objeto destino y el objeto enlace.
shape: Define una región seleccionable para hacer hipervínculos asociados con una forma asociada a una figura para crear un mapa. Los valores para este atributo son: circle, defaul, plygon y rect.
target: Valores especiales: "_blank" indica una nueva ventana, "_parent" indica el marco padre del marco que contiene el código fuente, "_self" indica el marco donde está el código fuente y "_top" indica la ventana completa del navegador.

Ejemplo

<a href="https://www.google.com">GOOGLE</a>

Resultado

GOOGLE

novalidate: Este atributo indica que el formulario no debe ser validado cuando se envíe.
target: Valores especiales: "_blank" indica una nueva ventana, "_parent" indica el marco padre del marco que contiene el código fuente, "_self" indica el marco donde está el código fuente y "_top" indica la ventana completa del navegador.
